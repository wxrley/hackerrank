Welcome to my [HackerRank](https://www.hackerrank.com/dashboard) challenge repository! \

## 💡 About the project
This repository contains solutions organized by [HackerRank](https://www.hackerrank.com/dashboard) challenges. \
Each folder represents a problem, and inside it, there is a `Solution.java` file with the solved code. \
The challenges vary in difficulty — from basic concepts to problems requiring more reasoning and structure.
