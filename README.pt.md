# 🧠 HackerRank Java Solutions
Bem-vindo ao meu repositório de desafios do [HackerRank](https://www.hackerrank.com/dashboard)! \
Aqui compartilho minhas soluções, escritas com foco em lógica e boas práticas.

## 💡 Sobre o projeto
Este repositório contém soluções organizadas por desafios do [HackerRank](https://www.hackerrank.com/dashboard). \
Cada pasta representa um problema, e dentro dela há o arquivo `Solution.java` com o código resolvido. \
Os desafios variam em dificuldade — de conceitos básicos até problemas que exigem mais raciocínio e estrutura.
